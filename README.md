# Customer Segmentation Analysis

## About This Project
I analyzed e-commerce customer data to group customers based on their buying behavior using RFM Analysis and KMeans Clustering.

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## What is RFM?
- **Recency** - How recently a customer purchased
- **Frequency** - How often they purchase
- **Monetary** - How much they spend

## What I Found
- Analyzed 500 customers
- Found 3 customer segments
- Top country by sales: Australia
- Best selling product: Knitted Blanket
- Total Revenue: Rs. 52,719.45
